Aurora Windows 使用说明

一、首次使用
1. 将整个 Aurora 文件夹解压到本地磁盘，例如 D:\Aurora
2. 进入 Aurora 文件夹
3. 双击 “启动 Aurora.bat”
4. 等待提示启动完成
5. 打开浏览器，访问 http://localhost:51880
6. 如果 Windows 弹出防火墙提示，请点击允许访问

二、平时常用
- 启动系统：双击 “启动 Aurora.bat”
- 停止系统：双击 “停止 Aurora.bat”
- 查看状态：双击 “查看状态.bat”
- 备份数据：双击 “备份数据.bat”

三、重要文件位置
- 程序文件：app\
- 配置文件：config\application.yaml
- 数据库文件：data\aurora.db
- 数据备份目录：data\backup\
- 日志目录：logs\

四、升级到新版本
建议按下面顺序操作：

1. 在旧版本文件夹里，双击 “停止 Aurora.bat”
2. 确认旧版本里的数据库文件还在：
   data\aurora.db
3. 将新版本压缩包解压到一个新的文件夹，例如 D:\Aurora-New
4. 用旧版本里的 data\aurora.db 覆盖新版本里的 data\aurora.db
5. 如果旧版本里有需要保留的配置，再检查 config\application.yaml 是否也要同步
6. 在新版本文件夹里，双击 “启动 Aurora.bat”
7. 浏览器打开 http://localhost:51880，确认可以正常登录和查看数据
8. 确认新版本没有问题后，再删除旧版本文件夹

五、请注意
- 不要手动删除 data\aurora.db
- 首次启动时，如果 Windows 弹出防火墙提示，请选择允许访问
- 如果启动失败，请先查看 logs\aurora-console.log、logs\aurora-error.log 和 logs\aurora.log
- 如果 51880 端口已被占用，请先关闭旧版本，或修改 config\application.yaml
